# Experimental / deferred — LLM-weight quantizer (docs)

These documents belong to STRAND's **experimental LLM-weight quantization track** — research,
design, and status notes for sub-4-bit model quantization, the `.strand` model-deploy formats,
GPU/Metal decode kernels, PV-tuning, GGUF comparisons, and model serving.

They are **out of scope for the core codec** and kept for reference only, alongside the rest of
the quarantined quant track (see [`../README.md`](../README.md)). Many describe designs that were
never built, were superseded, or were measured and set aside — a research record, not
documentation of the shipping codec.

The core codec's docs are the top-level [`README.md`](../../README.md) and the benchmark report
in [`../../docs/BENCHMARKS.md`](../../docs/BENCHMARKS.md).
