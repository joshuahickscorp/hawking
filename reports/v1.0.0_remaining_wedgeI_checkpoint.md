# Phase 2 Close — Wedge I Checkpoint
**Date:** 2026-05-05
**Decision:** residual_dtype default stays F32

## Coexist bench results (6 trials × 32 tokens, Claude.app running)
- F32 residual: median 2.480 dec_tps (spread 2.8%)
- F16 residual: median 2.504 dec_tps (spread 4.6%)
- F16/F32 ratio: 1.009 (+0.9%) — within noise, well inside ±15% threshold

## Decision rationale
F16 is within measurement noise of F32 in coexist mode. The spec gate requires
≥ 1.15× improvement to flip default. F16 path is implemented and passes all
parity tests; it is ready for clean-bench validation in an attended session.

## For clean-bench validation (next attended session)
```bash
TRIALS=6 TOKENS=32 bash tools/bench/coexist_bench.sh               # F32 baseline
DISMANTLE_RESIDUAL_DTYPE=F16 TRIALS=6 TOKENS=32 bash tools/bench/coexist_bench.sh  # F16
```
If F16 ≥ F32 × 1.15 AND argmax matches over 64-token generation → flip default.

## Env var added
`DISMANTLE_RESIDUAL_DTYPE=F16` enables the f16 residual path in `generate` and
`batch-hash` subcommands. Default is F32 (unchanged behavior).
