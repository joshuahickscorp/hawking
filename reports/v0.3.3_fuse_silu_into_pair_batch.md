Design:
- dispatch_gemv_q4_k_m_simd_pair_silu_batched encodes gate GEMV, up GEMV, and
  silu_mul into ONE ctx.dispatch_batch call (three sequential compute passes in
  one CommandBuffer). Metal serializes compute passes within a CB, so g_buf and
  u_buf are coherent when encode_silu_mul reads them.
- moe_expert_pair_matmul_dispatch now returns a: &mut [f32] (silu'd output)
  instead of separate out_a/out_b; both routed-expert loop and shared-expert
  block call the new function.
- CPU silu_mul is eliminated from the hot decode path for the simdgroup schedule.
